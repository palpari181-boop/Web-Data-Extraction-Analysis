## ✨ Features

- 📊 **Comprehensive Exploratory Data Analysis (EDA)** to uncover hidden patterns and trends in hotel data.
- ⭐ **Hotel Rating Analysis** to evaluate customer satisfaction and rating distribution.
- 💰 **Price Distribution Analysis** for understanding pricing patterns across hotels.
- 🏨 **Top 10 Most Expensive Hotels** visualization with detailed price comparison.
- 📦 **Box Plot Analysis** to detect pricing outliers and identify premium hotels.
- 🎯 **Price vs Rating Analysis** to explore the relationship between hotel pricing and customer ratings.
- 🔥 **Correlation Heatmap** for identifying relationships among numerical variables.
- 🥧 **City-wise Hotel Distribution** using interactive donut charts.
- 🌳 **Hierarchical Hotel Analysis** with Treemap visualization.
- ☀️ **Category-wise Analysis** using Sunburst charts.
- 🗺️ **Interactive Geographical Visualization** to analyze hotel locations across India.
- 📈 **Interactive Plotly Visualizations** with hover effects, zooming, and filtering.
- 🎨 **Luxury Dark Brown & Beige Theme** providing a premium and professional visual experience.
- 🧹 **Data Cleaning & Preprocessing** including handling missing values, duplicates, and inconsistent data.
- 📑 **Insight-Driven Analysis** with business-focused observations for every visualization.

## 🚀 Project Highlights

- ✔️ Cleaned and preprocessed real-world hotel dataset.
- ✔️ Created interactive visualizations using Plotly.
- ✔️ Applied statistical analysis to identify business insights.
- ✔️ Designed a premium visualization theme for better presentation.
- ✔️ Built as an end-to-end Data Analytics portfolio project using Python and Jupyter Notebook.
